# 🕒 Relógio Digital

[![Status](https://img.shields.io/badge/status-online-success?style=for-the-badge)](https://projeto-relogio-digital-teal.vercel.app/)

#### 🚀 Deploy do Projeto: [`https://projeto-relogio-digital-teal.vercel.app/`](https://projeto-relogio-digital-teal.vercel.app/)

## ✨ Visão geral

Um relógio digital simples que exibe horas, minutos e segundos, atualizando a cada 1 segundo. Projeto minimalista feito com HTML, CSS e JavaScript puro, usando uma fonte digital local para o visual clássico.
